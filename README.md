# tradingsystem
Django制作的一个交易平台

#####环境
**Python**:3.6
**Django**：1.11.7

#####实现的功能
**前端**：使用bootstrap制作前端页面
**后端**：使用Django搭建一个交易平台，要求实现商品展示、评价、购物车、订单导出等功能，使用xadmin做后台管理，整个项目框架搭建完成后加入商品图表分析等功能；
